# SF Displacement Forecast

Neighborhood-level analysis and forecasting of housing pressure in San Francisco using public data.

## Goals
- Build a clean neighborhood-year dataset
- Train models to predict next-year housing pressure
- Visualize results on an interactive map dashboard

## Repo Structure
- `data/raw/` raw downloads
- `data/processed/` cleaned datasets
- `src/` data + modeling code
- `app/` Streamlit dashboard
- `reports/` figures and writeups
